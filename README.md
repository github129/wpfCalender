# CalenderWpfApp
カレンダーを表示するWindowsアプリケーション

# システム要件
- windows10

# What is this?
指定した年のカレンダーを表示します。年の入力がない場合は当年のカレンダーを表示します。

# 主な機能
- １年分のカレンダーの表示。
- １か月viewと1年viewと切り替えることができます。
- オプションを操作することでviewが変更されます。
- 未入力の場合は当年のカレンダーを生成します。
- 年の入力は半角数字しか受け付けていません。それ以外はエラーとなります。

# オプション操作
- 【日曜日始まり】のチェックを外すことで、月曜日始まりのカレンダーにすることができます。
- 【当日の色】のチェックを外すことで、当日のBackgroundColorを無くすことができます。
